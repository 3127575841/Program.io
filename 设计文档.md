# 算法学习网站设计文档

## 一、配色方案

网站采用了简洁现代的配色方案，主要包含以下颜色：

1. 主色：蓝色 (#4A90E2) - 用于重点元素、按钮和强调部分
2. 辅助色：紫色 (#6D5DFC) - 用于悬停状态和次要元素
3. 强调色：红色 (#FF6B6B) - 用于特殊提示和强调内容

其他辅助色：
- 文本主色：深灰色 (#2C3E50)
- 文本浅色：中灰色 (#666666)
- 背景色：白色 (#FFFFFF)
- 背景浅色：浅灰色 (#F8F9FA)

## 二、布局设计

网站采用响应式布局，主要适配桌面端（≥1024px）。主要布局特点：

1. 顶部固定导航栏
   - Logo位于左侧
   - 导航菜单位于右侧
   - 在移动端会折叠为垂直排列

2. 全屏轮播图Banner
   - 高度为100vh（视口高度）
   - 包含左右切换按钮和底部指示点
   - 自动轮播功能

3. 内容区块采用卡片式布局
   - 每个部分都有明确的标题和内容区分
   - 课程区块采用网格布局
   - 练习区块采用分栏布局

4. 页脚区域
   - 简洁的设计风格
   - 包含版权信息和社交媒体图标

## 三、功能模块说明

1. 导航栏
   - 固定在顶部，方便用户随时导航
   - 菜单项包括：首页、算法课程、在线练习、模拟考试、管理员
   - 鼠标悬停效果：颜色变化
   - 点击菜单项实现平滑滚动到对应区域

2. 首页Banner
   - 包含3张全屏轮播图
   - 自动轮播功能（5秒切换一次）
   - 支持手动切换（左右按钮和底部指示点）
   - 轮播图内容包含标题和简短描述

3. 算法课程
   - 卡片式布局展示多个算法课程
   - 每个课程包含封面图、标题和简短描述
   - 提供"开始学习"按钮链接到详细课程页面
   - 分为热门课程、高级课程和特色课程三个区块

4. 在线练习
   - 网格布局展示三种难度的练习
   - 包括简单难度、中等难度和困难难度
   - 每个难度等级包含简短描述和开始练习按钮

5. 页脚
   - 网站标题和简短描述
   - 快速链接到主要功能区
   - 社交媒体图标（微信、微博、GitHub、知乎）
   - 版权信息

## 四、交互效果

1. 导航栏
   - 滚动时保持固定在顶部
   - 菜单项点击时平滑滚动到对应区域
   - 当前所在区域的菜单项高亮显示

2. 轮播图
   - 自动轮播效果
   - 支持手动切换
   - 切换时的淡入淡出过渡效果

3. 课程卡片
   - 鼠标悬停时上浮效果
   - 按钮颜色变化效果

4. 社交媒体图标
   - 悬停时上浮并变色效果

## 五、技术实现

1. HTML5
   - 语义化标签：header, section, footer等
   - 多媒体嵌入：img标签展示图片

2. CSS3
   - 弹性布局(Flexbox)和网格布局(Grid)
   - 过渡效果(transition)和变换(transform)
   - 动画效果(animation)
   - CSS变量定义全局样式

3. JavaScript
   - 轮播图自动切换功能
   - 平滑滚动导航
